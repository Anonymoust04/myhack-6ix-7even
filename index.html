MCP Servers — Governed Tool APIs

Model Context Protocol servers hosted on Cloud Run, governed by Apigee

● Running

Relationship MCP Server

Cloud Run
createLink
(mentorId, companyId, programmeId) → LinkEntity
Creates a governed, auditable relationship between ecosystem actors.
dissolveLink
(linkId, reason) → AuditRecord
Ends a relationship with full audit trail and reason logging.
getLinkHistory
(entityId) → LinkEntity[]
Returns all past and present relationships for any actor.
● Running

Scoring MCP Server

Cloud Run
scoreCompatibility
(profileA, profileB) → MatchScore
ML-powered compatibility score using Vertex AI embeddings.
rankCandidates
(entityId, candidatePool[]) → RankedList
Ranks a pool of candidates by fit, availability, and history.
● Running

Programme MCP Server

Cloud Run
getCapacity
(programmeId) → CapacityInfo
Returns current capacity, enrolled count, and waitlist.
enforceQuota
(regionId, limit) → QuotaStatus
Checks and enforces geographic or demographic quotas.
